Система компьютерного зрения для группового патрулирования БПЛА.

Нужно развернуть у себя виртуальное окружение и скачать зависимости 

## Установка
pip install -r requirements.txt
   
## Запуск   
Необходимо перейти в корневую папку проекта и вполнить команду:
    python process_frames.py
Затем выбрать режим в интерактивном меню
